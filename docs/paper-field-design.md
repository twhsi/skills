# Paper Field Design Direction

The twhsi.com public registry combines two identities:

1. BIRD supplies structure: address, index, route, deep link, black rules, and explicit information hierarchy.
2. The paper field supplies atmosphere: real folded objects, outdoor light, warm paper, saturated poster colors, organic marks, and a slower editorial rhythm.

The direction is informed by the public-facing energy of [Daisy Chain Fields](https://www.daisychainfields.com/): tactile paper surfaces, oversized retro display typography, documentary event photography, high-saturation pink/yellow/green/orange blocks, compact dark navigation, and card-based calls to action.

The adaptation remains original:

- use Yunghsi's real eight-page booklet photograph;
- retain the existing BIRD/twhsi palette and Rockwell-led typography;
- use circles and seed marks generated in CSS rather than copied floral graphics;
- retain the registry's BIRD, FIRE, iMandalArt, Skill Route, and machine-readable endpoint content;
- preserve accessibility, responsive layouts, and direct GitHub links;
- avoid copied logos, slogans, illustrations, or page composition.

The central publishing route is:

`Permanent Note → BIRD Address → BookletManifest → Folded Booklet`
