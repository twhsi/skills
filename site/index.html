iMandalArt 2.01

A compact 3x3 thinking card for LLM-assisted planning, writing, study, and decision work. Version 2.01 is currently CJK-friendly, with an English-native layout planned next.

GitHub

Featured Skill

Turn any source into one hard-line 3x3 card.

iMandalArt 2.01 asks an LLM to read the source first, identify one center idea, split it into eight orthogonal angles, and return an 11-line plain-text card that survives copy, paste, and note-app previews.

Weekly planning Compress health, writing, money, family, network, study, rest, and next action into one scan.
Writing focus Turn scattered notes into a center line plus eight non-overlapping editorial directions.
Knowledge capture Use a stable CJK hard-line format that works in TheBrain, Hermes, Discord, and plain notes.
