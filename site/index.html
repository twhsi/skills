Weekly Reverse Review

一個迭代近二十年的週檢視流程，轉成 AI-assisted Skill： 讀取 YEAR、Week、Day、Inbox 四個角度，再生成一週安靜而精彩的 8 Big Rocks。

GitHub

New skill

讓 AI 燒 tokens，人腦保留安靜判斷。

Weekly Reverse Review 會收集年計劃、百年人生計劃、上週計劃與週檢視、 七天日計劃與日記、行事曆，以及 inbox 裡的雜事。目標不是增加任務， 而是問：哪件事能帶來真正的幸福與寧靜？

腦力節省：11 個步驟從兩三小時變得更輕。 四個視角：YEAR、Week、Day、Inbox 先互相比對。 反向願望清單：變小、變慢、刪掉、延後，或留下陪伴。 核心答案：少說，陪伴，慢慢做。

iMandalArt 2.01

The featured skill in this registry: a hard-line 3x3 thinking card that lets mainstream LLMs turn loose notes, plans, and source material into one compact CJK-friendly Mandala index.

GitHub

Featured skill

One center, eight orthogonal angles, eleven physical lines.

iMandalArt 2.01 is optimized for CJK workflows today: each surrounding content phrase is exactly five Han characters, each title is compact, and full-width separators keep the card stable in chat previews, TheBrain/Cerebro notes, Hermes, Discord, and clipboard-based workflows.

The format is currently CJK-friendly. An English-native version is planned so the same 3x3 thinking rhythm can work naturally for English notes without forcing a Chinese character contract.
