# Weekly Eight-Domain Layout

Use this layout for weekly plans.

```text
Ⓕ去我內在　Ⓒ財流降噪　Ⓖ日文沉澱
...
...

Ⓑ書稿推進　◎◎◎◎◎　Ⓓ家人互動
...　幸福寧靜週　...
...　◎◎◎◎◎　...

Ⓔ人脈路徑　Ⓐ健康第一　Ⓗ開心休閒
...
...
```

## Domain Meanings

- `Ⓐ健康第一`: body, exercise, sleep, treatment, walking, badminton, running.
- `Ⓑ書稿推進`: writing, publishing, manuscript, course, agent skill, TheBrain, FIRE, iMandalArt, book cards.
- `Ⓒ財流降噪`: subscriptions, fees, income, cash flow, business plan, reducing money noise.
- `Ⓓ家人互動`: spouse, children, parents, siblings, family events, long companionship.
- `Ⓔ人脈路徑`: friends, collaborators, students, teachers, community, one-on-one meetings.
- `Ⓕ去我內在`: letting go, teacher stance, faith, silence, slowing down, not proving oneself.
- `Ⓖ日文沉澱`: Japanese, language study, sentence numbering, keyword index, learning logs.
- `Ⓗ開心休閒`: guitar, singing, nature, music, rest, hobbies, playful recovery.

## Phrase Style

- Use compact Chinese titles after labels.
- Prefer five-Han-character action lines.
- Keep `幸福寧靜週` as the center unless the user provides another center.
- Avoid turning every domain into a writing task.

## Minimum Action Pattern

When a domain is weak, give it one minimum action:

- Health: `羽球慢回身`, `跑走養節奏`, `早睡護心身`.
- Finance: `訂閱先盤點`, `現流養書路`, `費用先確認`.
- Leisure: `吉他聽太陽`, `合唱鬆心身`, `小樂不產品`.
- Family: `問夢不說教`, `陪走百年路`, `少說多陪伴`.
